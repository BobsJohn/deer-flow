# 能力广场服务层 — 产品定位（V1）

## 产品定义

能力广场服务层是 AI 能力的统一资产管理与分发平台。

1. **资产沉淀层**：将团队分散的 AI 能力（Skill、McpTool、SystemPrompt 等）沉淀为标准化、可复用的数字资产
2. **权限管控层**：按团队维度进行访问控制，核心资产支持加密保护
3. **能力分发层**：为执行侧（DeerFlow 等 Agent 框架）提供统一的发现、加载、执行、反馈闭环

## 设计原则

- **DeerFlow 原生优先**：能力广场不是独立系统，而是 DeerFlow 的"官方应用商店"
- **向后兼容**：现有 DeerFlow 项目中的 Skill 继续可用，支持渐进迁移
- **独立部署**：能力广场以独立服务部署（端口 8003），通过 REST API 与 DeerFlow 交互

## 用户角色

| 角色 | 典型用户 | 核心诉求 |
|------|---------|---------|
| 管理员 | 团队TL、平台负责人 | 掌控团队资产全貌、控制访问权限、审批核心资产发布 |
| 开发者 | 研发工程师、AI工程师 | 发布和管理自己的 Skill/Prompt/Tool |
| 使用者 | 业务方、其他研发团队 | 快速找到需要的 AI 能力，集成到自己的 Agent/业务中 |

## V1 范围

- 资产管理：Skill、McpTool、SystemPrompt、Agent、方案（Workflow）
- 扩展实体：PluginBundle、DigitalEmployee（来自产品方案）
- 权限模型：命名空间（团队）隔离 + 三档权限（公开/团队/商业）
- 加密策略：服务层解密后传递，V2 前不移至执行侧
