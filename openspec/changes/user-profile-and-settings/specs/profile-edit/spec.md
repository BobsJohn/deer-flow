# 个人信息编辑规格说明

## 新增需求

### 需求: 个人信息弹窗

#### 场景: 打开个人信息

- **当** 用户点击「个人信息」
- **那么** 弹出模态框，显示当前用户信息（头像、名称、邮箱、角色）
- **并且** 可切换 8 个内置头像
- **并且** 点击保存调用 `PUT /me` 更新

### 需求: 修改密码弹窗

#### 场景: 修改密码

- **当** 用户点击「修改密码」
- **那么** 弹出模态框，输入旧密码 + 新密码 + 确认新密码
- **并且** 新密码和确认密码不一致时提示错误
- **并且** 提交后调用 `PUT /me/password`
