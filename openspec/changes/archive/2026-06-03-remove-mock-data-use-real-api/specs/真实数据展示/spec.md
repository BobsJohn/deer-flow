# 规格说明

## 新增需求

### 需求: 市场首页从 API 获取技能列表

#### 场景: 页面加载成功且有数据

- **当** 用户访问 `/marketplace`
- **那么** 页面调用 `GET /api/marketplace/skills` 获取技能列表
- **并且** 渲染技能卡片网格

#### 场景: API 失败

- **当** API 返回非 200 状态
- **那么** 页面显示"加载失败，请重试"提示 + 重试按钮

### 需求: 技能详情页从 API 获取数据

#### 场景: 详情页加载成功

- **当** 用户访问 `/marketplace/skills/{id}`
- **那么** 页面调用 `GET /api/marketplace/skills/{id}` 获取技能详情
- **并且** 渲染完整详情页（含 Schema 和示例）

#### 场景: 技能不存在

- **当** API 返回 404
- **那么** 页面显示"技能未找到"提示

### 需求: 搜索调用后端 API

#### 场景: 用户输入搜索词

- **当** 用户在搜索框输入关键字并停留 300ms
- **那么** 页面调用 `GET /api/marketplace/search?q={keyword}`
- **并且** 展示搜索结果

#### 场景: 搜索无结果

- **当** 搜索 API 返回空列表
- **那么** 页面显示"没有匹配的技能" + 清除筛选按钮
